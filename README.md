# Finn Kvittering

Generate professional receipts and invoices from Finn.no listings. Perfect for creating accounting documents from Norwegian classified ads.

## Features

- Parse information directly from Finn.no URLs
- Generate professional PDF receipts and invoices
- Accounting-compliant documentation
- Privacy-first: all processing happens locally
- Clean, modern interface

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
